$ { project. WHY }
  문제정의 심화 팀 윈썸에 합류하여 알파랩중에 M랩을 가게 되었습니다. 저희 팀은 M랩을 신청하게된 이유는 저희 팀이 폐기물과 관련한 주제를 생각하고 있었고, 만들기에 관심이 더 많아서 M랩을 신청했습니다.
$ { project. HOW }
  M랩에서 정해준 일정으로 수업이 진행이 되었고, 최종 결과물로 수업 에서 배운 모든것을 활용해 저희 팀 문제정의 프로젝트와 연관된 결과물을 제작해야했습니다.
$ { project. WHAT }
  저희는 M랩 수업을 하면서 여러가지 메커니즘을 이용하여 많은 형태물을 만들어봤고, M랩 최종 결과물로 캔 압축기를 만들었습니다.
$ { self. development }
  생각보다 만드는것이 저의 적성에 맞았습니다. 의외로 재밌었고, 어렸을때 레고로 여러 모형을 만든 추억이 다시 떠오르기도 했습니다. 이번 M랩에서 팀원들과 함께 어떻게 무엇을 만들지 고민하고 의견을 나누어 최종 결과물까지 잘 만들수 있어서 너무 좋았습니다. M랩을 한줄 요약하자면 “실행착오를 통한 배움” 인것 같습니다. 수 없이 실패하고 또 수 없이 수정하고, 고치고 하면서 우리가 예상하지 못한 문제들을 하나씩 해결해나가는 과정이 좋았습니다.
