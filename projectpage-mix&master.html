  3모듈 개인주제프로젝트를 이어서 음악을 좀 더 깊이 배우고 싶은 마음이 생겨 음악을 만들때 빠질 수 없는 ‘믹싱’이라는 단계를 배우기로 했습니다. 저만의 음악을 더 완성도 높게 만들기 위해 믹싱을 배우기로 하였습니다.
$ { project. HOW }
  믹싱에 대해 아무 지식이 없어 ‘최이진의 믹싱과 마스터링’ 책을 중심으로 믹싱에 대해서 학습을 진행했습니다. 온라인 강의도 듣고, 인터넷에 있는 여려 자료들도 찾아봤습니다. 또한 음악 프로그램에 대해 좀 더 찾아보았고, 여러 장치에 대해, 또한 믹싱 작업이 무엇을 하는 작업인지 찾아봤습니다.
$ { project. WHAT }
  저는 이번 프로젝트를 하면서 음악을 만들때 사용할 수 있는 여러가지 장치들과 소리가 가지고 있는 특성, 믹싱의 기본, 여러가지 전문 용어를 새로 배우게 되었고, 결과물로 제가 프로젝트를 하면서 배운 여러가지 지식과 프로젝트를 어떻게 진행했는지 볼 수 있는 ebook을 만들었습니다. 직접 믹싱을 하기에는 믹싱이라는 작업이 전문성이 많이 필요한 작업으로 실습은 해보지 못했습니다.

$ { self. development }
  3모듈때보다 음악에대해 훨씬 많이 알게 되었습니다. 음악을 만들때 사용하는 여러 장치들의 작동 원리에 대해 더 알게되었고, 음악을 구성하고 있는 여러가지 요소를 어떻게 조율을 해서 듣기 좋게 만드는지에 대해 알게되었습니다. 저는 직접 음악을 만들 수 있는 단계까지 성장을 하였고, 제가 음악을 정말로 좋아한다는것을 한번더 느낄 수 있었습니다. 따라서 저의 장래 희망도 음악분야쪽으로 가고싶다는 생각이 들게 하였다.
