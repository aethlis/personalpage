$ { project. WHY }
  팀원들과 함께 브레인스토밍을 하다가 우연히 친환경 플라스틱 마크에 대한 영상을 보게 되었습니다. 조금 더 자세히 찾아보니 친환경 플라스틱의 종류는 다양하다는것을 알게되었고, 이 여러 가지 프라스틱이 갖고 있는 친환경적인 요소가 다 다르다는 것을 알게되었습니다. 친환경 플라스틱은 일반 플라스틱처럼 PET, PP, HDPE 등 플라스틱의 종류별로 표기를 하지않고, 다 통합하여 하나의 마크로 표기 하는것이 문제라고 생각하여 이 주제를 선정하게 되었습니다. 실제로 크게 친환경 플라스틱은 처리방법이 서로 다른 두가지 플라스틱이 있습니다. 생분해성 플라스틱과 태울때 직접적인 탄소 배출량을 줄이기 위한 바이오매스 플라스틱이 있습니다.
$ { project. HOW }
  국내와 해외 친환경 플라스틱 정의와 마크, 그리고 마크의 인증 기준들을 찾아보고 비교했습니다. 또한 친환경 플라스틱의 종류가 어떤것이 있고, 종류마다 가지고 있는 친환경적인 원리와 특성에 대해 학습했습니다. 일반 플라스틱의 제조과정과 친환경 프라스틱의 제조과정을 비교하고, 둘의 차이점에 대해서도 학습했습니다.
$ { project. WHAT }
