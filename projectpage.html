$ { project. WHY }
  당시 3모듈 대주제인 기후변화와 관련해 여러 기사와 뉴스를 보다가 기후변화가 주거환경에 줄 수 있는 영향과 관련한 뉴스를 보게되었습니다. 뉴스에서는 기후변화로 인해 생활의 터전이 파괴되면서 늘어나는 난민에 대한 내용을 전달하고 있었습니다. 저희팀은 인권과 관련하여 많은 관심이 있어 이러한 기후변화로 인한 주거환경 피해를 인권중에 ‘주거권’과 관련하여 기후변화와 연결시켜 팀 프로젝트를 진행하기로 했습니다. 생각보다 기후변화가 주거환경을 파괴함으로써 생겨나는 문제들이 많습니다. 생활 터전 파괴는 물론, 이외에도 이로 인한 환경난민 문제와 빈부격차 문제도 생겨난다는 것을 알 수 있었습니다.
$ { project. HOW }
  팀과 함께 주거권의 정의부터 시작해, 기후변화가 우리 주변 환경에 끼칠 수 있는 영향에 대해 찾아보았고, 그로 인해 생길 수 있는 피해사례들을 찾아봤습니다. 피해사례들에 관련해 수치화된 데이터와 자료를 모으고, 이번 모듈 대주제인 기후변화와 저희팀이 정한 주제인 주거권을 데이터와 자료를 근거로 삼아 연결시켜 기후변화가 인권에도 영향을 줄 수 있다는 사실을 알아냈습니다. 이후 학교 교과수업과 연결시켜 과목별로 저희팀 주제를 연결해봤습니다.
$ { project. WHAT }
  이번 프로젝트 결과물은 기본적으로 저희팀 주제와 저희가 진행한 프로젝트 내용들을 담은 ebook을 제작했고, 추가적으로 과목별로 과목에 맞게 여러 결과물을 제작하였습니다. 영어는 인포그래픽 형태로 기사를 제작하였고, 국어는 저희팀 주제와 관련해서 연설문을 작성해봤습니다. 수학은 데이터나 저희팀이 찾은 자료를 수치화하고 시각화하는 작업을 해봤습니다. 사회에서는 저희 주제를 인간중심주의적 시점과 생태중심주의적 시점으로 주제를 바라보는 경험을 해봤습니다.

$ { self. development }
  이번 프로젝트를 통해 기후변화가 앞으로 세상에 미칠 영향에 대해 생각해볼 수 있었습니다. 학습을 하게 되면서 기후변화가 무시할 수 없는 문제라는것을 깨닫게 되었습니다. 이번 주제 ‘기후변화가 주거권에 미치는 영향’에 대해 학습을 깊이 하면서 기후변화는 마치 도미노처럼 문제가 또 다른 문제를 일으킨다는것을 알 수 있었습니다. 특히 이번 학습을 통해 기후변화가 빈부격차와 난민문제까지 일으킬 수 있는 전세계적인 문제라는 것을 느낄 수 있었습니다.
